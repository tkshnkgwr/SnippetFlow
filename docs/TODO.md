# 開発状況およびタスク管理 (TODO.md)

本ドキュメントは、SnippetFlow（定型文クリップボード・マネージャー）における現在の実装状況（Done）、直近の課題（Todo）、および今後の機能拡張案（Backlog）を管理します。

---

## 1. 実装済みの機能一覧 (Done)

### 1.1. 共通基盤・UI設計
- **ハイブリッド動作環境**: Tauri版（React/TS + Rust）および egui版（純Rust）の双方が同一のコア機能を持つハイブリッド動作。
- **レスポンシブ・ダークモード**: Tailwind v4 (Tauri) および egui標準のビジュアルスタイルによる、ダークテーマとライトテーマの動的な切り替え。
- **固定ヘッダー・フッターレイアウト**: リストスクロール時にも操作部とアクションバーが固定される使いやすい画面設計。
- **多重起動防止 (Single Instance)**: 
  - Tauri: `tauri-plugin-single-instance` を採用し、重複起動時に既存ウィンドウにフォーカス。
  - egui: `single-instance` ミューテックスにより二重起動をサイレントに制限。
- **日本語文字化け（豆腐文字）防止**: Windowsシステムフォント（Meiryo 等）を起動時に動的ロードして自動登録。

### 1.2. 一覧画面 (List / Home)
- **スニペットカード表示**: 定型文をコンパクトなカード形式で一覧表示。
- **お気に入り（ピン留め）**: 「📌」マークによる最上部固定表示、およびIndigoカラーによる強調。
- **論理削除データの切り替え表示**: ゴミ箱内のデータを薄い赤色かつ打ち消し線付きでトグル表示。
- **多条件ソート**: 「更新日（新しい順/古い順）」「作成日（新しい順）」「タイトル順」「コピー数（よく使う順）」での並び替え。ソート順は永続化。
- **二重フィルター検索**: フリーワード検索（タイトル・本文・説明の部分一致）とタグ選択絞り込みの併用。
- **キーワードハイライト**: 検索ワードに一致するテキストを黄色背景でカラー強調表示。
- **タグクラウド**: データベースから一意なタグを自動抽出してトグルボタン化し、ワンクリックで絞り込み。
- **クイックコピー**: カード内の「📋 コピー」ボタンから即座にクリップボードへ格納（3秒間の完了トースト表示）。

### 1.3. 変更画面 (Form)
- **新規・編集処理**: 作成日時および更新日時の自動セット。編集時はIDやメタデータを明示表示。
- **論理削除・復元・物理削除**: 通常の削除（ゴミ箱移動）、アーカイブからの復元、データベースからの完全物理削除を段階的にサポート。
- **インテリジェント・タグ提案**: 入力中の「タイトル（重み2倍）」「本文」「説明」に含まれるワードと既存タグを分析し、おすすめタグを最大5つリアルタイム提示。

### 1.4. 差分比較画面 (Compare)
- **2カラム分割表示**: 選択した2つの定型文のタイトル・説明・本文プレビューを左右対称に並べて比較。
- **LCS（最長共通部分列）による行・文字単位の差分表示**: 追加箇所（緑背景）と削除箇所（赤背景）をカラーハイライト。
- **動的差し替えと左右入れ替え**: 画面遷移せずに上部のコンボボックスから対象を切り替え、または「⇄」で左右の立ち位置を反転。

### 1.5. 複数結合画面 (Merge)
- **スニペットの追加と除外**: チェックボックスでマージ対象をその場で動的に選別。
- **順序調整（↑ / ↓）**: 結合順序をボタン操作で上下に変更可能。
- **6種類の区切り文字選択**: 改行1つ、改行2つ、区切り線（`---`/`===`）、読点（`、`）、区切りなし。
- **ライブプレビューと一括コピー**: 結合結果をリアルタイムプレビューし、一括でクリップボードへコピー。

### 1.6. 性能メーター画面 (Performance)
- **データベース診断**: レコード数、推定JSONファイルサイズ、直近の検索クエリ実行時間（ミリ秒）を表示。
- **使用統計（アナリティクス）**:
  - コピー総回数と、累計のタイピング短縮時間（1文字＝0.3秒換算）の可視化。
  - コピー数が多い定型文トップ3のランキング表示。
- **ベンチマークテスト**: 検索処理を100回試行した際の平均処理時間の計測（ミリ秒）。
- **ダミーデータ負荷テスト**: 1,000件、2,000件、5,000件のダミーデータを一括生成・クリーンアップし、UIや検索の描画負荷を検証。
- **OS標準ダイアログによるバックアップと復元**: `rfd` クレートを使用した、安全なローカル JSON ファイルへのエクスポートおよびインポート。

## 2. 直近のタスク (Todo / In Progress)

### 2.1. Tauri版とegui版のデータ保存・設定の共通化
- [ ] **Tauriバックエンドへのデータ保存の完全移行**:
  現在、Tauri版はブラウザの `localStorage` にデータを保存し、egui版はローカルの `snippets.json` に保存している。
  Tauri起動時に Rust バックエンドを介してカレントディレクトリの `snippets.json` からデータをロードし、保存時も Rust 側でファイル書き込みを行う設計へアップグレードする。これにより、Tauri版とegui版でデータがリアルタイムに完全同期・共有される状態にする。
- [ ] **アプリ設定 (Settings) の完全一元化**:
  テーマ設定や選択したソート順などを、Tauri版の `localStorage` ではなく、egui版と同様に `settings.json` で一元管理できるようにする。

### 2.2. ロジックおよびテストの品質向上
- [ ] **共通ロジックのテストコード追加**:
  `common_lib` 内の LCS 差分計算や、インテリジェントタグ提案スコアリングについて、異常系や境界値（空テキスト、超長文など）のユニットテストを拡充する。
- [ ] **エラーハンドリングの強化**:
  ローカルの `snippets.json` ファイルが破損（JSONフォーマット不正）していた場合に、既存ファイルを即座に破壊せず、バックアップ（`.bak`）を作成した上でデフォルトロードする安全装置を Rust 側の `storage.rs` に実装する。

## 3. 今後拡張の余地がある機能の提案 (Backlog)

- [ ] **複数PC間のセキュア同期**:
  GitHub Gist や任意のオブジェクトストレージ、もしくはローカルネットワーク共有フォルダを介して、複数のPC端末間でスニペットデータを安全に自動同期する機能。
- [ ] **スニペットの暗号化保存**:
  個人情報やAPIキー、パスワードのテンプレートなどを安全に保管するために、ローカル保存される `snippets.json` を暗号化（AES-GCM等）するオプション機能。
- [ ] **グローバルホットキー (ショートカット呼び出し)**:
  アプリが最小化されている状態や非アクティブ状態であっても、特定のショートカットキー（例: `Ctrl + Shift + S`）で瞬時にウィンドウを最前面にポップアップ表示する常駐ユーティリティ化。
- [ ] **カテゴリ（フォルダ）階層化機能**:
  タグクラウドによる管理に加え、サイドバーにフォルダツリー（ビジネス、プライベート、開発用等）を表示し、定型文をカテゴリ階層で整理できる機能。
- [ ] **ペースト自動化（オートフィル）**:
  「コピー」ボタンをクリックした際、クリップボードに格納すると同時に自動でアプリウィンドウを非表示にし、直前までフォーカスが当たっていたアクティブウィンドウにテキストを自動ペースト（Ctrl+Vキーイベント送信）する機能。
