# 操作マニュアル・ユーザーガイド (USER_GUIDE.md)

本ドキュメントは、「定型文クリップボード・マネージャー (SnippetFlow)」の各機能の使い方と操作手順について解説するユーザーマニュアルです。

---

## 1. 基本操作と一覧画面 (定型文一覧 / Home)

アプリを起動すると最初に表示されるメインの画面です。登録されている定型文の検索やコピー、各種アクションへの遷移を行います。

### 1.1. インクリメンタルリアルタイム検索とハイライト
* **テキスト検索**: 最上部の検索窓にキーワードを入力すると、定型文の「タイトル」「本文」「説明」「ID」の部分一致でリアルタイムに絞り込みが行われます。
* **キーワードハイライト**: 検索キーワードと一致した文字列は、タイトルや本文内で黄色にカラーハイライト表示されます。
* **タグ検索**: 下部に並ぶタグクラウド（タグのリスト）から特定のタグをクリックすることで、トグルによる即時のタグ絞り込みが可能です。

### 1.2. クイックコピーと通知
* 各行の右側にある「📋 コピー」ボタンをクリックすると、定型文本文が即座にクリップボードにコピーされます。
* コピー完了時、画面上部（React版ではトースト通知、egui版ではヘッダー）に数秒間、完了した旨の通知が表示されます。

### 1.3. お気に入り（ピン留め）機能
* 各定型文カードに配置されたピン留めアイコン（📌）をクリックすると、その定型文をお気に入りに登録できます。
* ピン留めされた定型文は、**選択しているソート条件に関わらず常にリストの最上位に固定表示**されます。
* カード全体の境界線と背景がIndigo（薄い青）で強調表示されるため、頻繁に使用する定型文へのアクセスが格段に早くなります。

## 2. 変更画面 (新規定型文の登録・編集)

定型文の新規追加および、既存データの編集、削除を行います。

### 2.1. インテリジェント・タグ提案
* 定型文のタイトルや本文を入力すると、既存のスニペットで登録されているタグをリアルタイム分析し、関係性の深いタグをおすすめタグとして最大5つまで自動提示します。
* 提示されたタグをクリックするだけで、入力欄にタイピングすることなくワンタッチでタグを付与できます。

### 2.2. 論理削除・復元・物理削除
* **削除 (論理削除)**: 
  * 不要になった定型文を削除すると、ゴミ箱（過去ログアーカイブ）に移動し、通常の一覧からは非表示になります。
* **復元 / 永久削除**:
  * 一覧画面で「削除した定型文（過去ログ）を表示する」のチェックをONにすると、削除済みデータが打ち消し線付きでロードされます。
  * 編集画面から「🔄 アーカイブから復元」で元の状態に戻すか、「🗑️ 完全に削除する」でデータベースから物理的に永久排除するかを選択できます。

## 3. 高度な連携機能

### 3.1. 差分比較画面 (Compare)
* **概要**: ちょうど2つの定型文の差分を左右に並べて視覚的に比較する画面です。
* **操作**: 一覧画面でチェックボックスを用いて2つの定型文を選択し、「2件を比較」をクリックして遷移します。
* **LCS差分ビューアー**: 最長共通部分列(LCS)アルゴリズムに基づき、行ごとの追加されたテキスト（緑色背景）と削除・変更されたテキスト（赤色背景）をカラーハイライト表示します。
* **動的差し替え & 左右入れ替え**: 画面内のドロップダウンから比較対象をその場で変更したり、「⇄ 左右入れ替え」ボタンで配置を瞬時に入れ替えることができます。

### 3.2. 複数結合画面 (Merge)
* **概要**: 選択した複数の定型文を、任意の順序・区切り文字でマージして一括コピーする画面です。
* **順序調整**: 結合するスニペットをチェックボックスで動的に追加・除外でき、「↑」「↓」ボタンでマージされる順序を自在に変更できます。
* **区切り文字の選択**: 
  * 改行1つ / 改行2つ / 区切り線 (`---`) / 区切り線 (`===`) / 読点 (`、`) / 区切りなし の6種類から選択できます。
* **プレビュー**: リアルタイムでマージされた結果のプレビューテキストが表示され、「結果をコピー」ボタンで一括取得できます。

## 4. 性能メーターと使用統計（アナリティクス）

### 4.1. データベース・パフォーマンス診断
* **データベース件数**: 有効データ数、削除済みデータ数、推定JSON容量などを確認できます。
* **100回平均ベンチマーク**: 検索処理を100回連続で行い、ミリ秒単位でのリアルタイムな検索パフォーマンスを測定できます。
* **大量データ負荷テスト**: 一時的に「1,000件」「2,000件」「5,000件」のダミーデータを自動生成し、数千件のデータ下でもアプリが重くならず快適に動作することを体験・検証できます。

### 4.2. 使用統計 (アナリティクス)
* **総コピー回数**: アプリを通じてこれまでにコピーを行った累計回数をカウントします。
* **累計短縮時間**: コピーされた定型文の文字数に基づき、**「1文字書くのに0.3秒かかる」と仮定して、これまでにどれだけのタイピング時間を節約できたか**を「時間・分・秒」で可視化します。
* **よく使う定型文トップ3**: コピー回数の多い上位3つの定型文のランキング、コピー回数、それぞれのスニペットでの節約時間を表示します。

## 5. データバックアップ（インポート・エクスポート）

データを別のPCに移行したい場合や、手動でバックアップを取得したい場合に使用します。

* **エクスポート (📤)**: 現在のすべてのスニペットデータ（ピン留めやコピー統計情報含む）を `snippets.json` ファイルとしてローカルに保存します。
* **インポート (📥)**: 保存済みのJSONファイルを選択して読み込み、アプリのデータベースを過去のバックアップ状態に復元・上書きします。
